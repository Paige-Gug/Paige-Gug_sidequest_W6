## Project Title

Side Quest Week 6

## Author

Paige Gugeler

## Description

This game is designed to look like a forest where a rabbit must avoid dangers (foxes and boars) in order to collect all the keys. The sound effects and music are chosen to match the fun, but intense nature of the theme, adding to the game atmosphere.

---

## Interaction Instructions

Players will use the arrow keys to move around the screen and avoid boars, and use the space key to attack foxes, with the goal of collecting all 15 keys.
